# Course Repository
## Apples Game
### Start menu
Game has 3 modes: Infinity apples; Bonus speed; Acceleration boost;

**Description of game modes:**\
Infinity apples - apples are infinity and spawn in random position per eaten;\
Bonus speed - player's speed increased;\
Acceleration boost - player has accelerated per eaten apple.\

**Description of keys:**\
1 - Infinity apples (On/Off)\
2 - Acceleration boost (On/Off)\
3 - Bonus speed (On/Off)\
Up arrow - +1 apple\
Down arrow - -1 apple\
Left arrow - 20 apple\
Right arrow - 50 apple\
Escape - exit game\
Space - start game\

### Game
**Description of keys**\
Up arrow - turn up\
Down arrow - turn down\
Left arrow - turn left\
Right arrow - turn right\
Escape - finish game and open menu\
